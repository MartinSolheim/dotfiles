# Config files (ubuntu 17.10, i3-gaps, and polybar)


## scripts
#### bitFetcher
fetches crypto prices from coinmarketcaps api.  
```
./bitFetcher iota
```
returns the current iota price

no argument returns bitcoin price. a module for polybar is in the polybar config.


### dependencies
- feh 
- rofi
- compton
- curl
- jq
